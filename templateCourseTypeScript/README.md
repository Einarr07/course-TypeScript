# React + TypeScript Template

This project was created as a simple example to practice the commands needed to
start a React project with Vite, TypeScript, and SWC.

## Create the project

Run the Vite project generator:

```bash
pnpm create vite@latest
```

During the setup, use these options:

- Project name: `templateCourseTypeScript`
- Package name: `templatecoursetypescript`
- Framework: `React`
- Variant: `TypeScript + SWC`

## Install dependencies

Move into the project folder:

```bash
cd templateCourseTypeScript
```

Install the project dependencies:

```bash
pnpm install
```

## Run the project

Start the development server:

```bash
pnpm run dev
```

After running this command, Vite shows a local URL in the terminal. Open that URL
in the browser to see the React app.
